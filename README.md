# Siti Nuraeni Portfolio Site — v4

This version is redesigned to make the PNG pages the real main focus.

## What changed
- bigger image stage
- cleaner supporting text
- alternating editorial layout
- sticky portfolio images while scrolling
- progress dots on the side
- Apple-style calm spacing and rounded framing

## Add your PNG files
Export your PDF into PNG images and name them exactly:

- `page-01.png`
- `page-02.png`
- ...
- `page-13.png`

Put them inside:

```bash
assets/pages/
```

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/pages/`

## Publish to GitHub Pages
1. Create a GitHub repo
2. Upload all files
3. Commit
4. Go to Settings → Pages
5. Choose Deploy from a branch
6. Select `main` and `/root`
7. Save

## Easy edits
You can change highlighted pages and text in `script.js`.
